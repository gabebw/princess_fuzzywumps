# Adorable Pet Names

![](http://i.imgur.com/zTzCybb.png)

Inspired by the above image, this app randomly generates straight up _adorable_ pet names.
Examples:

* Baron Snugglesnout
* Princess Puddinwumps
* Count Jazzysnout

It runs on Padrino because I wanted to check out Padrino.

## Setup

Run `./bin/setup`.
